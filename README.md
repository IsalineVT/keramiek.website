# Keramiek Netlify + Decap CMS package

This package contains the Claude HTML website converted into a Netlify ready static website with an admin CMS.

## Fast publish on Netlify

1. Create a free Netlify account.
2. Drag this folder into Netlify Sites to deploy.
3. For CMS editing, connect the site to a GitHub repository.
4. In Netlify, go to Site configuration, Identity, Enable Identity.
5. Enable Git Gateway.
6. Invite admins under Identity.
7. Open /admin on your site.

## What admins can edit

Text, hero images, stats, about section, news, gallery, agenda, board, members, contact and footer are editable in /admin.

## Notes

The visual design is kept from the original Claude HTML. The existing static content remains as fallback if the JSON file cannot load.
